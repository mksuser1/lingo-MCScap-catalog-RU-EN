Комплект для начала работы с электродным шлемом.

Назначение

Для подготовки к ЭЭГ обследованию.

Состав комплекта

  • Гель электропроводящий,
  • шприц,
  • иглы одноразовые - 5 шт.,
  • щеточки для чистки электродов - 2 шт.,
  • спрей для дезинфекции
  • эксплуатационная документация.

Описание

Установочный комплект включает в себя все необходимое для начала работы с электродным шлемом: гель электродный контактный с высокой электропроводностью, шприц для геля, одноразовые иглы для заполнения электродов гелем через специальное отверстие в корпусе, щетки для очистки электродов и спрей для быстрой дезинфекции электродов, шлемов и принадлежностей.
