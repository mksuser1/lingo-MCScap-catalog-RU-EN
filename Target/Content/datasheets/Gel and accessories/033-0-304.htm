Комплект ярлыков для маркировки электродов по системе 10-20.

Состав комплекта

  • Ярлыки для электродов, символьные,

  • ярлыки для электродов, с цифрами.

Описание

Комплект ярлыков для маркировки электродов содержит парные клейкие ярлыки для крепления на проводе электрода. Один ярлык из пары предназначен для крепления рядом с электродом, второй — рядом с соединительным разъемом. Система маркировки ярлыков соответствует международной системе 10-20.
